# 🌿 GreenLens - 카메라 인식 기반 분리수거 안내 서비스

> **your recycling assistant**  
> 카메라로 촬영만 하면 분리수거 정보를 바로 확인할 수 있는 안드로이드 애플리케이션

<br>

## 📱 서비스 소개

일상에서 누구나 겪는 분리수거의 어려움을 카메라 인식 기술로 해결한 안드로이드 애플리케이션입니다.  
사용자가 재활용품을 촬영하면 YOLO 기반 객체 인식 모델이 자동으로 분류 정보를 안내해주며,  
복잡한 분리수거 기준을 몰라도 누구나 쉽고 빠르게 올바른 분리수거를 할 수 있도록 돕습니다.

- 🔗 **배포 링크**: [https://recycle-8z8j.onrender.com](https://recycle-8z8j.onrender.com)
- 🗂 **GitHub**: [https://github.com/wing853/recycle](https://github.com/wing853/recycle)

## 🏗 아키텍처

```
[사용자]
  ↓ 카메라 촬영
[Android 앱]
  ↓ 이미지 전송
[Spring Boot 서버] ←→ [MySQL DB]
  ↕ 이미지 전달 / 분석 결과 반환
[Flask AI 서버]
  ↓
[YOLO 모델 (best.pt)]

```

<br>

## 🤖 AI 모델

- **모델**: YOLOv8n-seg
- **데이터셋**: AI Hub 제공 이미지 약 20만장
- **분류 클래스**: 7개 (페트병, 플라스틱, 캔류, 유리병, 종이류, 비닐류, 스티로폼)
- **학습**: epoch 50, 로컬 GPU 환경
- **정확도**: 초기 65% → 최종 95~99% (클래스별)
- **폴백 처리**: YOLO 신뢰도 낮을 시 Gemini Vision API로 폴백

<br>

## ✨ 주요 기능

- 📷 **카메라 인식**: 재활용품 촬영 시 YOLO 모델이 자동 분류
- 📋 **분류 정보 안내**: 카테고리별 분리배출 방법 안내
- 🏆 **포인트 시스템**: 올바른 분리수거 시 포인트 적립 (3회 인식 실패 시 일반쓰레기 처리)
- 🛒 **포인트 스토어**: 적립된 포인트로 상품 교환 (카페, 식당, 편의점)
- 📊 **랭킹 시스템**: 포인트 기반 사용자 랭킹
- 👤 **소셜 로그인**: 네이버, 카카오, 구글 로그인 지원

## 🔥 트러블 슈팅

### Issue 1. JSON 양식 불일치로 인한 바운딩 박스 오매핑
- **문제**: AI Hub 라벨 JSON이 2가지 양식으로 구성되어 바운딩 박스가 잘못된 위치에 매핑
- **해결**: 양식별 분기 처리 로직 추가 (좌표 기반 / 폴리곤 기반 양식 분리 처리)

### Issue 2. 플라스틱 · 페트 · 유리 카테고리 오탐
- **문제**: 투명/반투명 외형이 유사한 3개 카테고리 간 혼동 발생
- **해결**: 해당 카테고리 데이터 재검토 및 반복 재학습으로 오탐률 감소

### Issue 3. Flask AI 서버 이미지 분석 불가
- **문제**: 팀 프로젝트 → 개인 프로젝트 전환 후 배포 환경 변경으로 AI 서버 동작 불가
- **해결**: Flask 서버 코드 전면 개선 (lazy loading, 모델 fallback 전략, 임시 파일 방식 이미지 처리)
